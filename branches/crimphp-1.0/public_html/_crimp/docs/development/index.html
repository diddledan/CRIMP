Development

This section of the documentation is to assist CRIMP users and developers in the creation of new CRIMP module plugins.

In general, there are several module types for Crimp:

  1. Generate content.
  2. Modify content.
  3. Background tasks.
  4. Form processing (not yet built into CRIMP)

These plugin modules are called from within the crimp.ini on either a global or per directory basis.

CRIMP is still under development, and so is this section :)
