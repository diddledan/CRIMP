CRIMP Requirements

To run CRIMP, please check that you can meet the following requirements:

  • A webhosting account based on the Apache webserver, or Apache installed on a machine you control. Other webservers may work, but are untested. The best way to get access to an Apache webserver is to sign up for Linux/UNIX WebHosting.
  • Webserver settings
    • Use of ".htaccess" files
    • Options FollowSymLinks OR Options SymLinksIfOwnerMatch
    • AllowOverride FileInfo (or preferably set to All if your host allows it)
  • PHP5 enabled on your selected webserver
  • For use of the CRIMP:PHP/Perl bridge, you require access to the perl binary on your system (usually in /usr/bin/perl on Linux hosts), along with the following Perl modules:
    • Fcntl
    • URI::Escape
    • Some CRIMPerl plugin modules may require additional perl modules in order to work.
