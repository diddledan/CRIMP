FTP Install

For this type of install, your downloaded module in either ZIP (.zip extension) or TARBALL (.tar.bz2 extension) format needs to be extracted into a temporary directory on your local machine before uploading to your web server.

From Windows

Using your favorite compression utility (example: Winzip) extract all the files to a temporary directory. Then open your FTP client and connect to your server host (or use windows file manager by typing 'ftp://[username]@[server-host.com]' into the address bar, you will be prompted for your password.). Select your extracted CRIMP directory and upload it in its entirety into your remote webserver home directory on your server host.
* please note, connection problems are not discussed in this document. Please check your firewall settings!

Once CRIMP is uploaded, you will need to check the file permissions on the uploaded files.

From Linux

  1. Untar the tarball into a temprary directory: 
    mkdir /tmp/crimp
tar -xjpvf Crimp-[VERSION].tar.bz2 -C /tmp/crimp
  2. Use your FTP client to upload the extracted CRIMP files into your remote webserver home directory.
  3. Once CRIMP is uploaded, you will need to check the file permissions on the uploaded files.
