CRIMP Installation

This is a Basic guide to installing CRIMP and assumes that your webserver is configured according to the default layout as shown within the CRIMP requirements documentation.

The CRIMP install should be unzipped or untared into your web server root, which is also your home directory. If your webserver or webhost is set up differently then please read the advanced installation documentation.

CRIMP Packaging

CRIMP is packaged in a compressed file and includes basic modules with an example configuration to get you started. These packages are usually provided in 2 formats, either a ZIP (.zip extension) or a TARBALL (.tar.bz2 extension). Download the appropriate one for your requirements depending on your type of install.
