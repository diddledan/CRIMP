500 - Server Error

A FATAL ERROR has occured, CRIMP cannot continue.
Please check the output of Debug View below.

Possible Reasons and Solutions

  1. This is a new install of CRIMP and the crimp.ini file is missing.
    • Rename the crimp.ini.example file to crimp.ini within the cgi-bin directory.

  2. The Config::Tiny perl module is missing.
    • Check your CRIMP install for Config/Tiny.pm within the cgi-bin directory.

  3. CRIMP has been incorrectly installed.
    • Visit the CRIMP Website for documentation and use the CRIMP mailing list if a solution cannot be found.
