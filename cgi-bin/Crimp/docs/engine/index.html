The CRIMP Engine

The main component in CRIMP is its engine,  a single file called index.pl within the cgi-bin of your website.

If you have problems with CRIMP, please FOLLOW THESE IMPORTANT STEPS!

Requirements

  • An account on one of the following webservers:
    Apache (linux)
  • A perl cgi-bin with the following perl modules:
    CGI
    Fcntl
  • File and directory permissions::
    cd ~
    chmod -v 755 cgi-bin/index.pl
    chmod -Rv 766 cgi-bin/Crimp/var

Initial setup

Point your browser to http://your-web-server/cgi-bin/index.pl

If this fails, then make sure the first line in 'cgi-bin/index.pl' is pointing to your perl executable.
#!perl
# windows path to perl (Default)
#!/usr/bin/perl
# linux path to perl
or create a symbolic link instead:
cd ~
ln -s /usr/bin/perl  cgi-bin/perl
