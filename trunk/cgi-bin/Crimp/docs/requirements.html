CRIMP Requirements

To run CRIMP, please check that you can meet the following requirements.
  • An account on one of the following webservers:
Apache (linux)
  • Webserver settings
AllowOverride (preferably set to All)
Use of ".htaccess" files
  • A perl cgi-bin with the following perl modules:
    (please note that some CRIMP plugin modules may require additional perl modules in order to work) 
CGI
Fcntl
