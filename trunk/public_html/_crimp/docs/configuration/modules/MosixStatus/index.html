MosixStatus

Synopsis:
crimp.ini 
MosixStatus = nodename1:id1,nodename2:id2...
crimp.xml 
<plugin name="MosixStatus" value="nodename1:id1,nodename2:id2..." />
Examples:
crimp.ini 
[/mosixstatus]
MosixStatus = neo:110,trinity:120,tank:130,dozer:140
crimp.xml 
<section name="/mosixstatus">
	<plugin name="MosixStatus" value="neo:110,trinity:120,tank:130,dozer:140" />
</section>
