ButtonBar

Description

This plugin module was originally intended for use with the default CRIMP setup and allows users to select view modes, view debug information, quickly access the CRIMP documentation and edit local content.

To use the CRIMP ButtonBar on your CRIMP site, just add it to the section you require within the crimp.ini file along with a minor edit to your template.

Installation

This module is included within the default CRIMP installation and only needs to be configured within the crimp.ini configuration file and within the users DocumentTemplate.

Configuration

This module can be used either globally or on a per directory basis and requires the DocumentTemplate module.

Edit crimp.ini for the following settings: 

# Global Options
ButtonBar = [Name of ButtonBar style]

# Per Directory Options
[/]
ButtonBar = [Name of ButtonBar style]

Edit your template with the following inside the <body> </body> section as to where you would like the ButtonBar to appear:

<!--BUTTONBAR-->

If the ButtonBar style does not exist, then ButtonBar = Default will be used.

Examples

This uses the default DocumentTemplate with the default ButtonBar for the docs section of your CRIMP website: 

[/docs]
DocumentTemplate = ../template/default-crimp.html
ButtonBar = Default

This uses a users custom DocumentTemplate with the BlackRounded ButtonBar (included with this module) in the global section of a CRIMP website: 

# Global Options
DocumentTemplate = ../template/myBlog.html
ButtonBar = BlackRounded
