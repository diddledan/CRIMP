500 - Server Error

A SEMI-FATAL ERROR has occured within the ContentDatabase plugin module.
The plugin has terminated, but CRIMP can continue operating. Please check the output of the Debug View if enabled (at the top right of your browser window).

Possible Reasons and Solutions

  1. CRIMP has been incorrectly configured for this plugin
    • Check that the required parameters are set in the configuration file.
  2. The DBI perl module is missing.
    • Check your Perl distribution for DBI.pm. you may need to install it from Yum, YaST, Apt, Portage, PPM, CPAN or whatever packaging system your platform uses.
  3. CRIMP has been incorrectly installed.
    • Visit the CRIMP Website for documentation and use the CRIMP mailing list if a solution cannot be found.
