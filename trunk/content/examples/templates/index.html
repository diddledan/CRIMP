Templates


CRIMP is able to handle templates created from a single HTML/XHTML document and this allows you to create templates in any style.
Some modules also make use of the <div id='crimp[modulename]><div> tags which enables you to move crimp content around on your page, set colours and fonts all by using CSS in the standard way.

To see different templates in action, simply select one from the Directory links above.
(please note, some templates will show directories and files that do not exist... these are for demonstration purposes)
